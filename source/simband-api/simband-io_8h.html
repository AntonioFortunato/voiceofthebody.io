Detailed Description

This file contains the public API for accessing SimBand data.

SimBand IO allows algorithms, sensor-modules and applications to interface with other sensor/algorithm data that resides on SimBand.

It contains functionality to send and receive data to streams, a timer allowing receival of data for several streams at the same time, and some API to discover streams.

Macro Definition Documentation

#define SIMBAND_STREAM_TYPE_ACCELEROMETER   "com.samsung.simband.accelerometer"

Accelerometer sensor signal.

To receive x, y, z for the accelerometer, use simband_io_stream_get_by_type_and_index and use 0, 1, 2 for the index respectively.

#define SIMBAND_STREAM_TYPE_BIO_IMPEDANCE   "com.samsung.simband.bioImpedance"

BioImpedance sensor signal.

Indices 0 and 1 correspond to the BioZ Q & T waves.

#define SIMBAND_STREAM_TYPE_BLOOD_PRESSURE   "com.samsung.simband.bloodPressure"

A stream representing a blood pressure reading.

The systolic and diastolic blood pressure can be accessed via indices 0 and 1.

#define SIMBAND_STREAM_TYPE_GSR   "com.samsung.simband.gsr"

galvanic skin response sensor signal

Indices 0 and 1 correspond to tonic and phasic GSR.

#define SIMBAND_STREAM_TYPE_PAT   "com.samsung.simband.pulseArrivalTime"

An algorithm representing pulse arrival time in seconds.

Pulse arrival time is the interval between the ECG pulse and the time PPG is detected.

#define SIMBAND_STREAM_TYPE_PPG   "com.samsung.simband.ppg"

PPG sensor signal (Photoplethysmogram)

There could be numerous PPG signals, each corresponding to a different LED/PD. Use simband_io_stream_get_by_type_and_index to discover the different PPGs.

simband_io_get_current_time

Parameters
[in]context
[out]timestampCurrent time in this context, in unix seconds.
Remarks
Note that the current time of the context is not the same as the application clock time. The context time is gated by the data that comes in via the streams of the context. So if the context was initialized with 3 streams, the clock will go forward only when those 3 streams have new data.
